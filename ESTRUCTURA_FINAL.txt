⚙️ SETUP RÁPIDO

1. Ambiente virtual:
   python -m venv venv && source venv/bin/activate

2. Instalar dependências:
   pip install -r backend/requirements.txt

3. Configurar .env:
   cp .env.example .env
   # Editar com credenciais PostgreSQL ou usar SQLite

4. Inicializar banco:
   python -m backend.cli db-init

5. Ingerir dados:
   python -m backend.cli ingest-prices -p 90d

6. Executar backend:
   python -m backend.app

7. Acessar API:
   http://localhost:8000/docs (Swagger UI)
   http://localhost:8000/redoc (ReDoc)

📝 NOTAS IMPORTANTES

1. PATHS
   - Todos os imports usam relative imports dentro do pacote backend
   - PYTHONPATH deve incluir: /sessions/.../market_platform_unified/

2. INGESTÃO
   - Rate-limit yfinance: 8 requisições/minuto
   - Checkpoint em: backend/logs/checkpoint.txt
   - Logs em: backend/logs/market_platform.log (rotativo 10MB)

3. BANCO
   - Suporta PostgreSQL (produção) e SQLite (desenvolvimento)
   - Todas as 9 tabelas criadas automaticamente no startup

4. CORS
   - Configurável via .env (CORS_ORIGINS)
   - Default: "*" (aceita qualquer origem)
   - Para produção: especificar origens

5. PERFORMANCE
   - Índices em: symbol, asset_id, date, country_id, sector_id
   - Constraints UNIQUE em: (asset_id, date), (company_id, period)
   - Paginação: máx 100 itens/page
