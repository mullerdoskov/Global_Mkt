ENDPOINTS CONSUMIDOS (9 TOTAL)
================================================================================

1. GET /api/prices/latest
   └─ Últimos preços de todos os ativos

2. GET /api/prices/latest?symbol=XXX
   └─ Preço de um símbolo específico

3. GET /api/prices/{symbol}/history?period={1S|1M|3M|6M|1A}
   └─ Histórico de preços com período

4. GET /api/assets/search?q={query}
   └─ Busca de ativos

5. GET /api/market/sectors
   └─ Dados agregados de setores

6. GET /api/fundamentals/{symbol}
   └─ Dados fundamentalistas

7. GET /api/fundamentals/{symbol}/valuation
   └─ Múltiplos de valuation

8. GET /api/ingestion/status
   └─ Status atual da ingestão

9. GET /api/ingestion/executions
   └─ Histórico de execuções

COMO USAR
================================================================================

1. Desenvolvimento Local:
   python -m http.server 8001
   open http://localhost:8001/index.html

2. Com Backend FastAPI:
   - Certifique-se que backend está rodando em http://localhost:8000
   - Frontend detectará automaticamente

3. Docker:
   Ver DEPLOYMENT.md para instruções

4. Servidor de Produção:
   Copiar index.html para servidor web

5. Cloud (Netlify, Vercel, Railway):
   Ver DEPLOYMENT.md para instruções
